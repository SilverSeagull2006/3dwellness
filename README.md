# 3dWellness

Личная ОС здоровья: Telegram Mini App + бот. Три измерения (биохимия / физика / психика), сигналы, дыхание, спокойный рассветный стиль.

## Структура
- `index.html` — экран «сегодня» (мини-апп). Помнит отметки (localStorage).
- `опросник.html` — опросник 1D (ЖКТ) → анализы → БАД → лист покупок.
- `bot/` — Telegram-бот: три сводки в день (утро/день/вечер). См. `bot/DEPLOY.md`.
- `стиль/` — референсы стиля.

## Запуск локально
Мини-апп: открыть `index.html` в браузере.
Бот: см. `bot/DEPLOY.md`.

## Деплой (чтобы работало 24/7 и на телефоне)
1. Мини-апп — на статический хостинг (GitHub Pages / Netlify), получить https-адрес → это `WEBAPP_URL`.
2. Бот — на Railway: переменные `BOT_TOKEN`, `WEBAPP_URL`, `TIMEZONE`; запуск `python bot.py` (worker).
