# 1D биохимия — разбор исходных опросников

Источник: анкеты нутрициолога Marina Vilkoff, RHN (Канада). Ответы Ольги 2014–2015 гг. — ИГНОРИРОВАТЬ (устарели). Файлы на английском → нужен перевод на русский (термины сверить с Ольгой).

## Что есть
1. **LAF — Lifestyle Assessment Form** (`2015_..._LAF.doc`) — большая входная анкета.
   Роль: слой **профиль/медпаспорт** + lifestyle-сигналы.
   Содержит: жалобы по приоритету; стресс 1–10 + источники; самопомощь; энергия; сон
   (часы, засыпание, качество); работа/смены; курение; рост/вес; **мед.история —
   лекарства, диагнозы, аллергии/анафилаксия, операции, камни**; **семейный анамнез**
   (аллергии, диабет, ЖКТ, алкоголизм, почки, артрит, ЖП, психзаболевания, астма,
   сердце, гипертония, остеопороз, кожа, рак, язвы); женский раздел (беременность,
   менструация, ПМС, менопауза, денситометрия); пищевые привычки (частота, продукты,
   напитки, жиры, тяга, реакции на еду).
   → Кормит: контраиндикации БАД (лекарства+диагнозы), приоритизацию анализов
   (возраст+семейный анамнез), сигналы (сон/стресс), 2D/3D.

2. **NSP — опросники по системам организма** (чек-листы симптомов).
   Оценка: **1 — лёгкий/редкий, 2 — умеренный/регулярный, 3 — выраженный/частый,
   пусто — не относится.** Сгруппированы по подсистемам.
   - NSP_1 Пищеварение: вялый желудок, печень, желчный, гиперацидный желудок,
     поджелудочная, дисгликемия
   - NSP_2 Кишечник: кандидоз, ПМС, паразиты
   - NSP_5 Лимфа/иммунитет: тимус, аллергии
   - NSP_8 Эндокринная: гипотиреоз, гипофиз, гипертиреоз, надпочечники
   - NSP_9 Опорно-двигательная: скелет, мышцы, нейромышечная
   - ПРОБЕЛЫ в нумерации: нет 3,4,6,7 (вероятно кровообращение/дыхание/
     мочевыделение/нервная) — уточнить, нужен ли полный набор для MVP.

3. **DEBQ** (`DEBQ_..._пищевого_поведения.pdf`, на русском) — типы нарушения пищевого
   поведения (ограничительное/эмоциогенное/экстернальное). Мост питание↔психика.

## Механизм (даёт этапы 1–3 конвейера)
`симптом (1–3) → сумма по подсистеме → подсистема «в фокусе» → добрать анализы + БАД`
Пороги в пилоте: доля от максимума <20% низкий, 20–45% умеренно (в фокусе),
>45% выраженно (в фокусе). Пороги — черновые, откалибровать.
