# Визуальный стиль 3dWellness

## Иллюстрации / иконки
**Одна непрерывная линия (one-line / монолайн).** Тонкий плавный росчерк, лёгкие петельки, ноль заливки. Референс — набор «одной линией» (Ольга прислала 2026-07-02; при желании положить сам файл сюда же).
- stroke-width ~1.8–2, `fill:none`, `stroke-linecap/linejoin: round`.
- цвет линии = тема/палитра (в приложении `currentColor`/переменные, чтобы адаптировалось к тёмной теме и красилось в hue блока).
- рисуем свои SVG под задачи (бад, питание, цикл, сон, забота, движение, напоминание, готово…), не стоковые.

## Палитра
Пастель, унисекс, спокойная: шалфей `#bfe3d3` / лаванда `#d3d0f2` / персик `#f3d5bd` / небо `#c4dcf3`. Фон тёплый off-white `#faf8f4`, чернила линий мягкие `#484843` (свет) / светлые (тёмная тема).

## Настроение / плотность
Спокойное, не игривое. Иконки — как акценты (по одной на секцию/карточку), не густо. Без лиц и мультяшности.

## Готовый набор монолайн-иконок (черновик)
капсула (бад) · лист (питание) · капля (цикл) · луна+искра (сон) · сердце (забота) · бегун (движение) · колокол (напоминание) · галочка-росчерк (готово). Показаны в чате, ждут апрув перед раскаткой по аппу.
